## 一、导入依赖
```
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-sleuth</artifactId>
</dependency>

<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-sleuth-zipkin</artifactId>
</dependency>

```

## 二、配置
```
zipkin:
    base-url: http://127.0.0.1:9411
  sleuth:
    sampler:
      probability: 1

```

# 第三方支付

采用的是第三方开源支付工具库IJPay。

官方文档:
https://javen205.gitee.io/ijpay/

接入指南:
https://javen205.gitee.io/ijpay/guide/

源代码:
https://github.com/javen205/IJPay


YC-Framework集成之后，只需引入如下包就能使用:
```
<dependency>
    <groupId>com.yc.framework</groupId>
    <artifactId>yc-common-pay</artifactId>
</dependency>

```

# 对象云存储
如果你不想维护分布式文件系统或单文件系统的话，可以使用公有云对象存储，YC-Framework支持国内主流的对象存储如阿里云、腾讯云、华为云、青云、百度云等。

使用其中任意的对象云存储，只需引入如下依赖即可:
```

<dependency>
    <groupId>com.yc.framework</groupId>
    <artifactId>yc-common-cs</artifactId>
</dependency>

```
